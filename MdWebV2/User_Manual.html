Luit.md User Manual.

This is Markdown editor where you can use html tags and latex ‘$’ delimiters. the area on the left hand side is the textbox area and on the left it is output area.

Some shortcuts:

  1. Opening File :- Ctrl+ O
  2. Rendering file :- Ctrl+ R
  3. Saving Opened File :- Ctrl+ S
  4. Saving As :- Ctrl+ Q
  5. Print :- Ctrl+ P
  6. Reading View :- Ctrl+ L

Luit.md is an simple easy to use text editor on web. As your files are stored on your system there is no fear of privacy. The print function downloads the text as HTML file.
